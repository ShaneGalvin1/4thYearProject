About Us

Mobile Application:

  • Available on Android Platform.
  • Users can register using the Android Application.
  • Once registered, users can record scores for a match they are watching.
  • Optionally if the user is involved with a team they may choose to record statistics for a chosen team in the match.


Web Application:

  • The live scores feature of the Winning Edge Web Application allows anyone to view live scores and results from G.A.A. matches recorded today.
  • If a registered user of the Android Application, a user may choose to login.
  • Once signed in, a user is brought to their home page, where and statistics they have recorded will be displayed.
  • Statistics are displayed in a manner which allows the user to gain maximum benefit and provide feedback to the team.
